---------------------------------------------------------------------------------------------------
Шаг: 1
    composer require tymon/jwt-auth
---------------------------------------------------------------------------------------------------
Шаг: 2
    npm install
---------------------------------------------------------------------------------------------------
Шаг: 3
    Проверить наличие строки в файле config/app.php
        'providers' => [
                ...
                Tymon\JWTAuth\Providers\LaravelServiceProvider::class,
        ]
    Шаг: 3.1
        Если есть, то переходим на Шаг: 4
    Шаг: 3.2
        Если нет, то добавить строку:
            Tymon\JWTAuth\Providers\LaravelServiceProvider::class,
        Затем нужно опубликовать конфиг
            php artisan vendor:publish --provider="Tymon\JWTAuth\Providers\LaravelServiceProvider"
---------------------------------------------------------------------------------------------------
Шаг: 4
    Сгенерировать секретный ключ
        php artisan jwt:secret
---------------------------------------------------------------------------------------------------
Шаг:5
    В .env после JWT_SECRET добавить:
    JWT_TTL=10080
    Это время жизни токена в минутах
---------------------------------------------------------------------------------------------------
При проблемах
    1. Удалить vendor
    2. Удалить node_modules
    3. Удалить composer.lock
    4. Удалить package-lock.json
    5. composer install
    6. npm install
    7. npm run dev
---------------------------------------------------------------------------------------------------
